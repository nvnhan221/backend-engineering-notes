# Examples

This directory contains runnable code examples that complement the documentation.

## Structure

```
examples/
├── api/              # API examples
├── database/         # Database examples
├── auth/             # Authentication examples
└── utils/            # Utility examples
```

## Usage

Each example includes:

- Complete, runnable code
- README with setup instructions
- Dependencies (if any)
- Expected output

## Running Examples

1. Navigate to the specific example directory
2. Read the README for setup instructions
3. Install dependencies (if needed)
4. Run the example

## Contributing

When adding new examples:

1. Create a new directory under the appropriate category
2. Include a README with:
   - Description
   - Prerequisites
   - Setup instructions
   - How to run
   - Expected output
3. Keep code well-commented
4. Follow the existing example structure
